# Emberchain Wallet

A standalone cryptocurrency wallet application built with React, Vite, and TailwindCSS.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is configured for Vercel deployment. Simply connect this repository to Vercel and it will build automatically.

## Tech Stack

- React 18
- Vite 5
- TailwindCSS 4
- Radix UI components
- Wouter for routing
- React Query for state management
